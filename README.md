# Run Locally

Clone repository

git clone https://github.com/your-username/professional-resume.git

cd professional-resume

Install dependencies

npm install

Start development server

npm run dev

Open

http://localhost:5173

---

# Build

npm run build

Preview

npm run preview

---

# Deploy

npm run deploy

Then enable GitHub Pages

Settings → Pages

Branch: gh-pages
